# 网腾无限AI 圣经箴言

## 项目介绍

**网腾无限AI 圣经箴言** 是一款深受 nomads.com (Nomad List) 数据看板风格启发、融合圣经经言智慧与现代灵修解压文化的数字代祷测评微应用。该应用基于 Vue 3 + Vite + Vanilla CSS 打造，用户能对自身的灵性维度（信心、平安、爱心、智慧、试炼）进行滑动评分并倾诉代祷需求。AI 会精算并呈现“自我打分 vs AI共识”的对比看板，并按选定流派（含使徒箴言默想、九巨擘神学论等）生成圣言与灵修代祷解析。同时，页面内置了支持 Web Audio 灵性圣钟发声与恩典增长的互动祈祷圣烛。

### 核心特性
- **灵性五维评分交互**：信心指数 (Faith)、平安指数 (Peace)、爱心慈悲 (Love)、智慧感悟 (Wisdom)、试炼困惑 (Trial) 的无缝滑动打分。
- **互动祈祷圣烛 (Virtual Prayer Candle)**：前端利用 Web Audio API 动态合成 528Hz 空灵教堂圣钟声，点击圣烛即可累加恩典祷告计数并浮现“恩典+1”渐隐动画。
- **AI 判定与灵性对比看板**：自动提取 AI 回复末尾的共识数据，并在前端渲染并排的双轨数据对比图表。
- **圣经箴言卡式历史记录**：将所有代祷箴言与恩典计数持久化于本地，以精美的“圣经箴言卡”网格形式展示，支持详情回填、单条删除与一键清空。
- **右上角常驻分享**：磨砂玻璃悬浮分享按钮，快速呼出微信朋友圈分享指引。
- **并排二维码打赏与联系弹窗**：打赏栏并排展示微信/支付宝支付码，Contact Us 弹窗并排展示微信/钉钉联系码。
- **无下划线自适应弹窗**：条款及隐私弹窗支持 max-height 限高内滚动，而二维码弹窗高度自适应，防止双滚动条。

## 快速启动

### 1. 克隆项目
```bash
git clone https://github.com/WT-Agent/ai-bible.git
cd ai-bible
```

### 2. 安装依赖
项目强制使用 pnpm 作为包管理器：
```bash
pnpm install
```

### 3. 配置本地开发环境
复制并配置环境变量：
```bash
cp .env.example .env
```
在 `.env` 中填入您的 API 密钥：
- `DEEPSEEK_API_KEY`: 您的 DeepSeek 开发者 API 密钥（用于文本生成任务）

### 4. 开发与构建
启动本地开发服务（支持本地反向代理中转，防止前端密钥泄露）：
```bash
pnpm dev
```
打包生产静态资源（打包输出在 `dist/` 目录中，支持零成本部署于 GitHub Pages、Vercel 或 OSS 容器）：
```bash
pnpm build
```
